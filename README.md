## Overview

This project aims to develop and deploy a **new frontend service** for the DEI Private Cloud platform, replacing the existing PHP-based solution that presents scalability issues and an outdated user interface.

The new solution follows a modern **Frontend/Backend architecture** with:
- **Frontend**: Single Page Application (SPA) built with **Angular**
- **Backend**: REST API built with **Node.js**
- **Authentication**: Integration with institutional **LDAP**
- **Real-time updates**: WebSocket-based communication
